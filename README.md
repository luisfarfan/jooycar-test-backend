### Jooycar backend test

## Levantar localmente

1. Instalar las dependencias del proyecto

```bash
npm install
```

2. Correr el proyecto

```bash
npm run dev
```

## Levantar con Docker

1. Construir la imagen

```bash
docker-compose up --build
```

La api esta configurada para correr en 'http://localhost:8080/api/v1/trips/'
